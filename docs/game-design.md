# Game Design Outline

## Genre
3D Action RPG

## Pitch
A high-mobility action RPG where players restore shattered realms by defeating region guardians and reclaiming lost memory shards.

## Player Fantasy
- Stylish melee/magic combos
- Meaningful build choices
- Exploration-driven upgrades

## Combat Loop
1. Scout enemy pack
2. Engage with opener (dash strike / spell)
3. Weave combo + dodge timings
4. Spend resource on finisher
5. Loot and upgrade

## Progression Systems
- Character Level + Attribute points
- Weapon rarities and perk affixes
- Skill Nodes (active + passive)
- Relic slots for build-defining modifiers

## Enemy Design Tiers
- Trash mobs: positioning pressure
- Elites: mechanic checks (armor break, interrupt windows)
- Bosses: multi-phase move sets

## World Structure
- Hub: crafting, quests, NPC progression
- Regions: linear-critical path with optional challenge branches
- Dungeons: short, mechanic-focused combat gauntlets

## MVP Scope
- One class
- One region
- Three standard enemies
- One boss
- Inventory + save/load
